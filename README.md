# 🏦 Bank Performanc and Risk Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents a **Bank Risk Analysis Dashboard** built using **Power BI** to analyze customer financial behavior and identify potential risk patterns.
The dashboard helps in understanding customer income distribution, loan behavior, and financial indicators that can help banks assess potential risks and make data-driven decisions.

The analysis focuses on uncovering trends related to **customer demographics, financial attributes, and loan characteristics**.

---

## 🎯 Objectives

* Analyze customer financial data to identify **risk indicators**
* Understand **income distribution and customer profiles**
* Identify patterns in **loan amount and financial attributes**
* Monitor the **performance of top 5 banks in India**
* Create an **interactive dashboard** for easy exploration of insights

---

## 🛠 Tools & Technologies Used

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX** – Calculated measures and metrics
* **Excel / CSV Dataset** – Data source

---

## 📈 Dashboard Insights

The dashboard provides insights such as:

* Distribution of **customer income**
* Analysis of **loan amounts**
* Financial behavior patterns
* Identification of **high-risk customer segments**
* **Performance** of top 5 banks
* Comparison of financial attributes across different categories

These insights help financial institutions make **better risk management decisions**.
